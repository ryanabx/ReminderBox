# Contributing

This document has information on how to compile ReminderBox.

## Dependencies

- Rust
- Cargo
- Trunk
    
    You can install trunk in many ways.

    Install trunk through cargo:
    
    ```sh
    cargo install trunk
    ```

    Ref: https://book.leptos.dev/getting_started/index.html

## Testing locally

```sh
trunk serve --open
```

## Building

```sh
trunk build --release
```
